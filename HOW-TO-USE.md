**Step 2: Log in to Sanity**

**Step 3: Add Artwork**
1. Click **"Artwork"** in the left sidebar
2. Click **"Create new document"** (+ button)
3. Fill in the form:
   - **Title**: Name of the artwork (e.g., "Sunset Dreams")
   - **Slug**: Click "Generate" (auto-creates URL-friendly version)
   - **Image**:
     - Click "Select image"
     - **On phone**: "Take photo" or "Choose from photos"
     - **On computer**: "Browse" to select file
   - **Description**: Optional story about the piece
   - **Medium**: e.g., "Oil on canvas", "Watercolor", "Acrylic"
   - **Dimensions**: e.g., "24" √ó 36""
   - **Year**: When it was created
   - **Display Order**: Lower number = shows first (0, 1, 2, etc.)
4. Click **"Publish"**
5. **Done!** The artwork appears on the website immediately

## üí° Pro Tips

### For Best Photo Quality:
- Use good lighting (natural light is best)
- Take photos straight-on (not at an angle)
- Higher resolution = better (phones usually take great photos!)
- Don't worry about file size - Sanity optimizes automatically

### Organization:
- Use **Display Order** numbers: 0 (first), 1 (second), 2 (third), etc.
- Update **Year** to group by time period
- **Featured** toggle: Can be used for highlighting special pieces

### Quick Edits:
- Click any artwork in Studio to edit it
- Change the order, update descriptions, swap photos
- Always hit "Publish" to save changes

## üÜò Troubleshooting

**"Can't upload images"**
- Check file format (JPG, PNG, WEBP work best)
- Try compressing very large files (over 10MB)
- Refresh the page and try again

**"Changes don't show on website"**
- Hard refresh the website: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
- Wait ~10 seconds for deployment to update

**"Forgot to click Publish"**
- Go back to the document in Studio
- Click "Publish" button
- That's it!
